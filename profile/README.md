## Getting Started
1. **Clone** the game repo  
   ```bash
   git clone https://github.com/Studio-Todos/Paradox.git
   cd game
   ```
2. **Build & Run**  
   ```bash
   cargo run --release
   ```
3. **Join the Discord** for dev chat, feedback, and pairing up on issues!
